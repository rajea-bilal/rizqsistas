# Images Directory

This directory contains all images for the Rizq Sistas website.

## Directory Structure

- `hero/` - Hero section images (2 images needed)
- `values/` - Value proposition section images (4 images needed)  
- `events/` - Events section images (3 images needed)
- `about/` - About section images (2 images needed)
- `blog/` - Blog/resources section images (3 images needed)

## Image Requirements

### Hero Section
- `hero-member.jpg` - Portrait of Rizq Sistas member
- `hero-collaboration.jpg` - Collaborative work scene

### Values Section  
- `education.jpg` - Education/learning scene
- `community.jpg` - Community gathering
- `tools.jpg` - Financial tools/planning
- `network.jpg` - Networking/growth

### Events Section
- `workshop.jpg` - Workshop/seminar setting
- `discussion.jpg` - Group discussion
- `planning.jpg` - Financial planning scene

### About Section
- `digital-tools.jpg` - Digital tools/technology
- `office-space.jpg` - Professional office environment

### Blog Section
- `investment.jpg` - Investment/finance related
- `balance.jpg` - Work-life balance
- `budgeting.jpg` - Budgeting/planning

## Image Specifications

- **Format**: JPG or PNG preferred
- **Size**: Minimum 1200px wide for optimal quality
- **Aspect Ratios**:
  - Hero images: 16:9 or similar
  - Value images: 4:3 ratio
  - Event images: Square (1:1)
  - About images: 3:4 (portrait)
  - Blog images: Square (1:1)

## Usage

Images should be placed in their respective folders and referenced in the code using `/images/[folder]/[filename]`.
